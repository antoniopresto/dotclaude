## Exportar Conversas do Claude Code

**Não existe comando nativo para exportar conversas automaticamente**, mas há alternativas:

### Opções Disponíveis

**1. Modo Headless com JSON**
```bash
claude -p "sua pergunta" --output-format json > conversa.json
```

**2. Redirecionamento simples para texto**
```bash
claude -p "sua pergunta" > conversa.txt
```

**3. Acumular múltiplas interações**
```bash
claude -p "pergunta 1" >> historico.txt
claude -p "pergunta 2" >> historico.txt
```

**4. Resumo da sessão atual**
```
/summary
```
Depois copie o resultado para um arquivo.

**5. Processar JSON estruturado**
```bash
claude -p "pergunta" --output-format json | jq . > conversa.json
```

### Para Sessões Interativas

As conversas ficam armazenadas internamente. Você pode:
- Usar `--resume` para continuar sessões anteriores
- Copiar manualmente do terminal (`Cmd+C` no macOS)
