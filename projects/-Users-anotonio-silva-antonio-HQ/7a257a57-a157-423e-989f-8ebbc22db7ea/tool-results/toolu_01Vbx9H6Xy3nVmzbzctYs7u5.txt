# Split Layout Component

## Overview
The Split layout is a PatternFly component designed for "horizontal layouts with spacing." It enables flexible content arrangement across rows.

## Key Features

**Basic Functionality:**
The component renders children in a horizontal split arrangement, with optional spacing and wrapping capabilities.

**Props for Split:**
- `hasGutter` (boolean, default: false) - "Adds space between children"
- `isWrappable` (boolean, default: false) - "Allows children to wrap"
- `children` - Content to render
- `component` - Base element (defaults to div)
- `className` - Additional CSS classes

**Props for SplitItem:**
- `isFilled` (boolean, default: false) - "Flag indicating if this split layout item should fill the available horizontal space"
- `children` - Item content
- `component` - Base element (defaults to div)

## Styling

The component uses CSS custom properties for consistent spacing. The gutter gap defaults to `1rem` (tied to PatternFly's spacing tokens).

## Usage Examples

Three example patterns are provided:
1. **Basic** - Simple horizontal split
2. **With gutter** - Spacing between items
3. **Wrappable** - Content that wraps to multiple lines
