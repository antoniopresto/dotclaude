# Stack Layout Component

## Overview
The Stack layout is a React component designed for arranging content vertically. According to the documentation, it "arranges content in a vertical stack" and provides flexible spacing options.

## Key Features

**Basic Properties:**
- **Component**: Renders as a `<div>` by default but accepts custom elements
- **Children**: Accepts React nodes for content placement
- **Gutter Support**: Optional `hasGutter` boolean prop that "adds space between children"

**StackItem Properties:**
- **Fill Behavior**: The `isFilled` prop allows items to "fill the available vertical space"
- **Customization**: Supports className and custom component rendering

## Spacing Configuration

The stack uses CSS variables for spacing management:
- Default gutter gap: `1rem`
- Variable references include `--pf-t--global--spacer--md` and `--pf-t--global--spacer--300`

## Practical Usage

Basic implementation uses simple children rendering, while the "with gutter" example demonstrates spacing between stacked elements. The component pairs with `StackItem` for fine-grained control over individual sections within the layout.
