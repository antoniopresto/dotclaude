# Flex Layout Component Overview

## Core Purpose
The Flex layout provides a responsive, CSS flexbox-based system for arranging content with comprehensive alignment, spacing, and distribution controls.

## Key Spacing Systems

**Two spacing approaches:**
- **Spacing system**: "applies a margin between the flex layout's direct children to create space between items"
- **Gap spacing**: Uses CSS `gap` for rows (`rowGap`), columns (`columnGap`), or both (`gap`)

## Essential Props for Layout Control

### Alignment Props
- `alignItems`: Controls alignment along the cross axis (flexStart, flexEnd, center, stretch, baseline)
- `alignSelf`: Individual item alignment override
- `alignContent`: Multi-line flex alignment
- `align`: Applies margin auto for positioning at breakpoints

### Spacing Props
- `spaceItems`: Uniform spacing for all children (none, xs, sm, md, lg, xl, 2xl, 3xl, 4xl)
- `spacer`: Individual item spacing control
- `gap`/`rowGap`/`columnGap`: CSS gap-based spacing with same size options

### Distribution Props
- `justifyContent`: Controls main-axis distribution (flexStart, flexEnd, center, spaceBetween, spaceAround, spaceEvenly)
- `direction`: Sets flex direction (row, column, rowReverse, columnReverse)
- `flexWrap`: Controls wrapping behavior (wrap, nowrap, wrapReverse)

### Flexibility Props
- `flex`: Controls flex growth (flexDefault, flexNone, flex_1-4)
- `grow`: Enables flex growth at breakpoints
- `shrink`: Enables flex shrinking
- `fullWidth`: Sets width 100% at specified breakpoints

### Responsive Control
All props support breakpoint-based values (default, sm, md, lg, xl, 2xl).
