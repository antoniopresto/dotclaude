## OrthographicView Overview

The OrthographicView class is a subclass of View that creates a top-down view of the XY plane. It is usually used for rendering 2D charts in non-geospatial use cases.

## Key Features for Timeline/Chart Visualizations

### Setting Up OrthographicView
If using non-geospatial data, you will need to manually create a view that is appropriate for info-vis, using imports like `import {Deck, OrthographicView} from '@deck.gl/core'`.

### Zoom and Axis Control
The zoom prop of the OrthographicView now supports an `[x, y]` Array to specify independent zoom levels. A zoomAxis option is added to the OrthographicController to restrict zooming to the X or Y axes.

### Additional Options
OrthographicView adds a flipY option.

### 64-bit Projection Support
OrthographicView and OrbitView now also support 64-bit projection, with no extra code changes required. This greatly improves the visual quality when rendering very large and/or dense graphs.

## Time-Based Animations
There are examples of using DeckGL with time-based animations, such as creating animated visualizations of NYC Taxi trip data that show how data on a map changes over time.
