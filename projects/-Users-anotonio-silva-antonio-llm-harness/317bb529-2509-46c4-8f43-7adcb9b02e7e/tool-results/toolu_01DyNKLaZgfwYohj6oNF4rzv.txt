### Claude Agent SDK (Python)
Claude Agent SDK for Python is the official Python SDK from Anthropic for programmatic interaction with Claude Code. It wraps the Claude Code CLI and provides a concise, Pythonic API that supports bidirectional conversation, custom tools, hook system, permission control, and async-first cross-platform async support.(no content)The SDK spawns the CLI as a subprocess; the CLI talks to the Anthropic API.(no content)The Claude Code CLI is automatically bundled with the package - no separate installation required! ### Programmatic Tool Calling (API Feature)
Programmatic tool calling allows Claude to write code that calls your tools programmatically within a code execution container, rather than requiring round trips through the model for each tool invocation. This reduces latency for multi-tool workflows and decreases token consumption.(no content)Programmatic tool calling is currently in public beta. To use this feature, add the "advanced-tool-use-2025-11-20" beta header to your API requests.
