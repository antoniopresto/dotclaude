   126→## Codebase Patterns
   127→
   128→### Domain Services
   129→- Location: `src/services/`
   130→- Pattern: Class with injected dependencies
   131→- All services emit OpenTelemetry traces
   132→- Use Drizzle ORM for database access
   133→
   134→### API Routes
   135→- Location: `src/api/routes/`
   136→- Framework: Hono
   137→- All routes have OpenAPI spec
   138→- Use @belt/tools createSchema for validation
   139→
   140→### Workers
   141→- Location: `src/workers/`
   142→- Framework: BullMQ
   143→- Each worker is a separate file
   144→- Concurrency defined in worker config
   145→
   146→### Optimizer IPC
   147→- Protocol: NDJSON over stdin/stdout
   148→- Input: `{ type: "solve", data: {...} }`
   149→- Output: `{ type: "progress" | "solution" | "error", ... }`
   150→
   151→### Validation & State Management (@belt/tools)
   152→
   153→> **CRITICAL**: See README.md for complete @belt/tools API reference
   154→
   155→**Schema Validation** (replaces Zod):
   156→```typescript
   157→import { createSchema, createType, InferType } from '@belt/tools';
   158→
   159→const TrainModel = createSchema({
   160→  id: 'string',
   161→  trainNumber: 'string',
   162→  mileageKm: 'number',
   163→  fitnessStatus: { enum: ['FIT', 'RESTRICTED', 'GROUNDED'] },
   164→});
   165→type Train = InferType<typeof TrainModel>;
   166→const train = TrainModel.parse(data);
   167→```
   168→
   169→**State Management** (replaces React hooks + zustand):
   170→```typescript
   171→import { createSlice, createMachine } from '@belt/tools';
   172→
   173→const fleetSlice = createSlice({
   174→  id: 'fleet',
   175→  initial: { trains: [], selected: null },
   176→  cache: { policy: 'session' },
   177→  reducers: {
   178→    setTrains: (state, trains) => { state.trains = trains; },
   179→  },
   180→});
   181→
   182→const machine = createMachine({
   183→  id: 'mms',
   184→  slices: { fleet: fleetSlice },
   185→  trackMetadata: false, // Critical for 60fps
   186→});
   187→
   188→// In React component (NO useState/useEffect!)
   189→const { state, actions } = machine.useMachine('fleet');
   190→```
