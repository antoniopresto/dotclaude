   911→### F122: Fix elastic/jump effect when dragging - COMPLETE
   912→
   913→**Problem:**

   915→

   917→React state updates are batched/async. When user starts a new drag while momentum is running:

   926→   - `handleCameraInitialized`

   930→   - `handlePointerMove` (pan movement)
