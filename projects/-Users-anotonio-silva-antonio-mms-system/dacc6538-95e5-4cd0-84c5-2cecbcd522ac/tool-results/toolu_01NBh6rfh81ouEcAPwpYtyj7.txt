   128â†’### Domain Services
   129â†’- Location: `src/services/`
   130â†’- Pattern: Class with injected dependencies

   134â†’### API Routes
   135â†’- Location: `src/api/routes/`
   136â†’- Framework: Hono

   140â†’### Workers
   141â†’- Location: `src/workers/`
