     1→# Harness Orchestrator - AI Instructions
     2→
     3→## What is This?
     4→
     5→This is a **sub-project** within the MMS monorepo. It's an AI orchestration system that manages multiple Claude agents to complete harness tasks in parallel.
     6→
     7→**IMPORTANT**: This is isolated from the main MMS canvas project. Do NOT modify files in `apps/canvas/` when working on this package.
     8→
     9→---
    10→
    11→## Harness Protocol
    12→
    13→```
    14→1. READ progress.txt → understand current state
    15→2. READ features.json → find next feature (passes: false)
    16→3. IMPLEMENT the feature
    17→4. TEST the implementation
    18→5. UPDATE features.json → set passes: true
    19→6. UPDATE progress.txt → add session notes
    20→7. COMMIT changes to this package only
    21→```

    68→## Key Algorithms
    69→
    70→### 1. Dependency Graph (DAG)
    71→
    72→```typescript
    73→// Build graph from features
    74→features.forEach(f => {
    75→  graph.addNode(f.id);
    76→  if (f.blockedBy) {
    77→    const blockers = Array.isArray(f.blockedBy) ? f.blockedBy : [f.blockedBy];
    78→    blockers.forEach(b => graph.addEdge(b, f.id));
    79→  }
    80→});
    81→
    82→// Find ready tasks (no pending deps)
    83→function getReadyTasks() {
    84→  return features.filter(f =>
    85→    !f.passes &&
    86→    getDependencies(f).every(dep => dep.passes)
    87→  );
    88→}
    89→```
    90→
    91→### 2. Critical Path
    92→
    93→```typescript
    94→// Longest path determines minimum completion time
    95→function criticalPath(graph) {
    96→  const distances = topologicalSort(graph).reduce((acc, node) => {
    97→    const maxPredecessor = Math.max(
    98→      0,
    99→      ...getPredecessors(node).map(p => acc[p] + 1)
   100→    );
   101→    acc[node] = maxPredecessor;
   102→    return acc;
   103→  }, {});
   104→
   105→  return Object.entries(distances)
   106→    .sort((a, b) => b[1] - a[1]);
   107→}
   108→```
   109→
   110→### 3. File Locking
   111→
   112→```typescript
   113→// Prevent race conditions on features.json
   114→import { lockfile } from 'proper-lockfile';
   115→
   116→async function atomicUpdate(path, updater) {
   117→  const release = await lockfile.lock(path);
   118→  try {
   119→    const data = JSON.parse(readFileSync(path));
   120→    const updated = updater(data);
   121→    writeFileSync(path, JSON.stringify(updated, null, 2));
   122→  } finally {
   123→    release();
   124→  }
   125→}
   126→```
