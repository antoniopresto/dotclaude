   137â†’## Codebase Patterns
   138â†’
   139â†’### Domain Services
   140â†’- Location: `src/services/`
   141â†’- Pattern: Class with injected dependencies
   142â†’- All services emit OpenTelemetry traces
   143â†’- Use Drizzle ORM for database access
   144â†’
   145â†’### API Routes
   146â†’- Location: `src/api/routes/`
   147â†’- Framework: Hono
   148â†’- All routes have OpenAPI spec
   149â†’- Use @belt/tools createSchema for validation
   150â†’
   151â†’### Workers
   152â†’- Location: `src/workers/`
   153â†’- Framework: BullMQ (Redis-backed)
   154â†’- Central exports: `src/workers/index.ts`
   155â†’- Use `registerWorker()` to create new workers
   156â†’- 8 queues: ingestion, tactical_scheduler, strategic_planner, optimizer, manhour_leveling, drift_monitor, notification, report
   157â†’
