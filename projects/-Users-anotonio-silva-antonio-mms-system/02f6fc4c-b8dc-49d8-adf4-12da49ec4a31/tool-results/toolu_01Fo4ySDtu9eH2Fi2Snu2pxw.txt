   115→-----------------------------------------
   116→O playbook de Geoffrey Huntley inclui guardrail 99999999999:
   117→
   118→"When @IMPLEMENTATION_PLAN.md becomes large, periodically clean out

   125→----------------------------------------------------------------------------
   126→Fonte: https://github.com/snarktank/ralph
   127→
   128→"Ralph automatically archives previous runs when you start a new feature
   129→(different branchName). Archives are saved to archive/YYYY-MM-DD-feature-name/"
   130→
   131→Arquivos de estado:
   132→- prd.json - Status de conclusão de tasks
   133→- progress.txt - Acumulador de learnings (append-only)
   134→- Git commits - O histórico permanente
   135→
   136→ESTRATÉGIA 3: Padrão "Fresh Context"
   137→------------------------------------
   138→Fonte: HumanLayer ACE-FCA (github.com/humanlayer/advanced-context-engineering-for-coding-agents)
   139→
   140→"Each iteration clears context. Re-read specs, plan, code every cycle.
   141→Fresh context is reliability."
   142→
   143→O arquivo de plano deve conter APENAS trabalho ativo. Fases completas vão para:
   144→- Histórico git (commits são seu changelog)
   145→- Arquivos de archive separados
   146→- Ou simplesmente deletados (git tem o histórico)
   147→
   148→ESTRATÉGIA 4: Consciência de Budget de Tokens
   149→---------------------------------------------
   150→Fonte: https://github.com/agrimsingh/ralph-wiggum-cursor
   151→
   152→WARN_THRESHOLD=70000   # Enviar aviso de wrap-up
   153→ROTATE_THRESHOLD=80000 # Forçar rotação de contexto
   154→
   155→Implementações Ralph rotacionam contexto a 40% da capacidade (80k/200k),
   156→não a 90%. Isso é intencionalmente "wasteful" mas garante qualidade.

   288→5.1 SCRIPT: Pre-Iteration Context Budget Check
   289→--------------------------------------------------------------------------------
   290→Arquivo: ralph/scripts/check-context.sh
   291→
   292→#!/bin/bash
   293→# Verifica tamanho dos arquivos antes de cada iteração
   294→
   295→check_file_size() {
   296→    local file="$1"
   297→    local max_tokens="${2:-15000}"
   298→    local tokens=$(wc -w < "$file" | awk '{print int($1 * 1.3)}')
   299→
   300→    if [[ $tokens -gt $max_tokens ]]; then
   301→        echo "WARNING: $file has ~$tokens tokens (max: $max_tokens)"
   302→        echo "Consider running: ralph plan --cleanup"
   303→        return 1
   304→    fi
   305→}
   306→
   307→check_file_size "ralph/IMPLEMENTATION_PLAN.md" 15000
   308→check_file_size "ralph/AGENTS.md" 2000
   309→
   310→5.2 SCRIPT: Automatic Archiving Hook (Post-Commit)
   311→--------------------------------------------------------------------------------
   312→Arquivo: ralph/scripts/archive-completed.sh
   313→
   314→#!/bin/bash
   315→# Arquiva tasks completas após commits
   316→
   317→archive_completed_phase() {
   318→    local plan="ralph/IMPLEMENTATION_PLAN.md"
   319→    local archive_dir="ralph/archive/$(date +%Y-%m-%d)"
   320→
   321→    # Contar completas vs total
   322→    local completed=$(grep -c "^\s*\[x\]" "$plan" || echo 0)
   323→    local total=$(grep -c "^\s*\[.\]" "$plan" || echo 0)
   324→
   325→    if [[ $completed -gt 50 ]]; then
   326→        mkdir -p "$archive_dir"
   327→
   328→        # Extrair seções completas para archive
   329→        grep -B2 "^\s*\[x\]" "$plan" >> "$archive_dir/completed.md"
   330→
   331→        echo "Archived $completed completed tasks to $archive_dir/"
   332→        echo "Run 'ralph plan --refresh' to regenerate clean plan"
   333→    fi
   334→}
   335→
   336→archive_completed_phase
   337→
   338→5.3 SCRIPT: Plan Regeneration
   339→--------------------------------------------------------------------------------
   340→Arquivo: ralph/scripts/refresh-plan.sh
   341→
   342→#!/bin/bash
   343→# Regenera plano dos specs (descarta cruft acumulado)
   344→
   345→ARCHIVE_DIR="ralph/archive/$(date +%Y-%m-%d)"
   346→mkdir -p "$ARCHIVE_DIR"
   347→
   348→# Arquivar plano atual
   349→cp ralph/IMPLEMENTATION_PLAN.md "$ARCHIVE_DIR/pre-refresh.md"
   350→
   351→# Git é o histórico real
   352→git log --oneline --since="30 days ago" > "$ARCHIVE_DIR/recent-commits.log"
   353→
   354→# Regenerar plano fresh
   355→./ralph/loop.sh plan 3  # Max 3 iterações para planning
   356→
   357→echo "Plan refreshed. Archive: $ARCHIVE_DIR/"
