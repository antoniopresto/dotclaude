Mathematical Formulas\n   479→\n   480→### 4.1 Km Variance (Coefficient of Variation)\n   481→\n   482→Primary metric for fleet mileage balance. Lower variance = better distribution.\n   483→\n   484→```\n   485→Km Variance % = (Standard Deviation / Mean) * 100\n   486→\n   487→Where:\n   488→  mean = sum(all train km) / n\n   489→  stddev = sqrt(sum((x_i - mean)^2) / n)\n   490→```\n   491→\n   492→```typescript\n   493→function calculateKmVariance(kmValues: number[]): number {\n   494→  if (kmValues.length < 2) return 0;\n   495→\n   496→  const mean = kmValues.reduce((a, b) => a + b, 0) / kmValues.length;\n   497→  if (mean === 0) return 0;\n   498→\n   499→  const squaredDiffs = kmValues.map(km => Math.pow(km - mean, 2));\n   500→  const variance = squaredDiffs.reduce((a, b) => a + b, 0) / kmValues.length;\n   501→  const stdDev = Math.sqrt(variance);\n   502→\n   503→  return (stdDev / mean) * 100;\n   504→}\n   505→```\n   506→\n   507→| Status | Variance | Indicator |\n   508→|--------|----------|-----------|\n   509→| Green (Healthy) | <= 10% | Green badge |\n   510→| Yellow (Warning) | 10% - 15% | Yellow badge |\n   511→| Red (Critical) | > 15% | Red badge |\n   512→\n   513→---\n   514→\n   515→### 4.2 Fleet Availability\n   516→\n   517→```\n   518→Fleet Availability % = (Fit Trains / Total Trains) * 100\n   519→\n   520→Targets:\n   521→  Target: >= 97%\n   522→  Critical: < 95%\n   523→```\n   524→\n   525→```typescript\n   526→function calculateFleetAvailability(fitTrains: number, totalTrains: number): number {\n   527→  if (totalTrains === 0) return 0;\n   528→  return (fitTrains / totalTrains) * 100;\n   529→}\n   530→```\n   531→\n   532→---\n   533→\n   534→### 4.3 MTBF / MTTR / MDBF\n   535→\n   536→```\n   537→MTBF = Total Operating Hours / Number of Failures (Hours)\n   538→MDBF = Total Operating km / Number of Failures (Kilometers)\n   539→MTTR = Total Downtime Hours / Number of Repairs (Hours)\n   540→       Target: <= 24 hours, Critical: > 48 hours\n   541→```\n   542→\n   543→```typescript\n   544→interface FailureMetrics {\n   545→  mtbf: number;\n   546→  mdbf: number;\n   547→  mttr: number;\n   548→}\n   549→\n   550→function calculateFailureMetrics(\n   551→  operatingHours: number,\n   552→  operatingKm: number,\n   553→  downtimeHours: number,\n   554→  failureCount: number,\n   555→  repairCount: number\n   556→): FailureMetrics {\n   557→  return {\n   558→    mtbf: failureCount > 0 ? operatingHours / failureCount : Infinity,\n   559→    mdbf: failureCount > 0 ? operatingKm / failureCount : Infinity,\n   560→    mttr: repairCount > 0 ? downtimeHours / repairCount : 0,\n   561→  };\n   562→}\n   563→```\n   564→\n   565→---\n   566→\n   567→### 4.4 Coordinate  635→```\n   636→deviation_pct = |current_km - target_km| / max(target_km, 1) * 100\n   637→\n   638→If current_km < target_km:\n   639→  penalty = -deviation_pct * 10  (BONUS)\n   640→If current_km >= target_km:\n   641→  penalty = +deviation_pct * 10  (PENALTY)\n   642→```\n   643→\n   644→```typescript\n   645→function calculateKmPenalty(\n   646→  currentKm: number,\n   647→  targetKm: number,\n   648→  fleetAvgKm: number\n   649→): number {\n   650→  const effectiveTarget = targetKm || fleetAvgKm;\n   651→  if (effectiveTarget === 0) return 0;\n   652→\n   653→  const deviationPct = Math.abs(currentKm - effectiveTarget) / effectiveTarget * 100;\n   654→\n   655→  if (currentKm < effectiveTarget) {\n   656→    return -Math.round(deviationPct * 10);\n   657→  } else {\n   658→    return Math.round(deviationPct * 10);\n   659→  }\n   660→}\n   661→```\n   662→\n   663→---\n   664→\n   665→### 4.6 Haversine Distance (Dead Running)\n   666→\n   667→Great circle distance between two geographic points.\n   668→\n   669→```\n   670→a = sin^2(delta_lat/2) + cos(lat1) * cos(lat2) * sin^2(delta_lon/2)\n   671→c = 2 * atan2(sqrt(a), sqrt(1-a))\n   672→distance = R * c\n   673→\n   674→Where: R = 6371 km (Earth\'s radius)\n   675→```\n   676→\n   677→```typescript\n   678→function haversineDistance(\n   679→  lat1: number, lon1: number,\n   680→  lat2: number, lon2: number\n   681→): number {\n   682→  const R = 6371;\n   683→  const toRad = (deg: number) => deg * Math.PI / 180;\n   684→\n   685→  const lat1Rad = toRad(lat1);\n   686→  const lat2Rad = toRad(lat2);\n   687→  const deltaLat = toRad(lat2 - lat1);\n   688→  const deltaLon = toRad(lon2 - lon1);\n   689→\n   690→  const a = Math.sin(deltaLat / 2) ** 2 +\n   691→            Math.cos(lat1Rad) * Math.cos(lat2Rad) *\n   692→            Math.sin(deltaLon / 2) ** 2;\n   693→\n   694→  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));\n   695→\n   696→  return R * c;\n   697→}\n   698→```\n   699→\n Array.filter() is FORBIDDEN.**\n  1432→\n  1433→```typescript\n  1434→import RBush from \'rbush\';\n  1435→\n  1436→interface RBushItem {\n  1437→  minX: number; minY: number;\n  1438→  maxX: number; maxY: number;\n  1439→  id: string;\n  1440→}\n  1441→\n  1442→export class RBushSpatialIndex {\n  1443→  private tree: RBush<RBushItem>;\n  1444→  private itemsById: Map<string, RBushItem> = new Map();\n  1445→\n  1446→  constructor() { this.tree = new RBush(); }\n  1447→\n  1448→  load(tasks: TaskRender[]): void {\n  1449→    this.tree.clear();\n  1450→    this.itemsById.clear();\n  1451→\n  1452→    const items: RBushItem[] = tasks.map(task => ({\n  1453→      minX: task.startDay * PX_PER_DAY,\n  1454→      maxX: (task.startDay + task.duration) * PX_PER_DAY,\n  1455→      minY: -task.trainIndex * PX_PER_TRAIN - OVERSCAN,\n  1471→      maxX: bounds.maxX + OVERSCAN,\n  1472→      minY: bounds.minY - OVERSCAN,\n  1473→      maxY: bounds.maxY + OVERSCAN,\n  1474→    });\n  1475→    return results.map(item => item.id);\n  1476→  }\n  1477→\n  1478→  queryNearest(x: number, y: number, maxDistance = 50): { id: string; distance: number } | null {\n  1479→    let radius = 20;\n  1480→    let nearest: { id: string; distance: number } | null = null;\n  1481→\n  1482→    while (radius <= maxDistance) {\n  1483→      const candidates = this.tree.search({\n  1484→        minX: x -
