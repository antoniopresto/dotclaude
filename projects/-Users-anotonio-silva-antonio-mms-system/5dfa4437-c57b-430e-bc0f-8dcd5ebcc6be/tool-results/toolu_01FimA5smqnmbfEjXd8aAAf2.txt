   162â†’## Codebase Patterns
   163â†’
   164â†’### Domain Services
   165â†’- Location: `src/services/`
   166â†’- Pattern: Class with injected dependencies
   167â†’- All services emit OpenTelemetry traces
   168â†’- Use Drizzle ORM for database access
   169â†’
   170â†’### API Routes
   171â†’- Location: `src/api/routes/`
   172â†’- Framework: Hono
   173â†’- All routes have OpenAPI spec
   174â†’- Use @belt/tools createSchema for validation
   175â†’
   176â†’### Workers
   177â†’- Location: `src/workers/`
   178â†’- Framework: BullMQ (Redis-backed)
   179â†’- Central exports: `src/workers/index.ts`
   180â†’- Use `registerWorker()` to create new workers
   181â†’- 8 queues: ingestion, tactical_scheduler, strategic_planner, optimizer, manhour_leveling, drift_monitor, notification, report
   182â†’
   183â†’**Running Workers:**
   184â†’```bash
