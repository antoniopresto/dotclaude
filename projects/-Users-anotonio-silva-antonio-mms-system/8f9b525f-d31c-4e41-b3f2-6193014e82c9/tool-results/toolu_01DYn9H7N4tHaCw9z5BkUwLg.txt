## What is the Ralph Wiggum Technique?

Ralph Wiggum is a loop. You give Claude Code a task, it works on it, and when it tries to exit, a Stop hook blocks the exit and feeds the same prompt back in. The files it just changed are still there, so each iteration can build on the last one.(no content)The Ralph Wiggum Technique, created by Geoff Huntley, went viral in the final weeks of 2025.
