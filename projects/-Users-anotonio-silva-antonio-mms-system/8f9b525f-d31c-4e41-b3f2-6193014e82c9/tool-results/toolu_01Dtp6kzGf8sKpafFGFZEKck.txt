# The Ralph Technique for Claude Code

## Core Concept
Ralph is a iterative automation method utilizing a simple Bash loop:

```bash
while :; do cat PROMPT.md | claude-code ; done
```

This technique continuously feeds prompts to Claude Code without tool call limitations, creating a deterministic yet adaptable workflow.

## Philosophy & Approach

**Deterministic Imperfection**: "That's the beauty of Ralph - the technique is deterministically bad in an undeterministic world." The method acknowledges inherent defects while treating them as tunable rather than insurmountable.

**Iterative Refinement**: Rather than abandoning failed attempts, users "tune Ralph like a guitar." When the system takes wrong directions, the operator adjusts prompts with contextual guidance—similar to adding instructional signage to redirect behavior.

**Skill-Based Mastery**: Ralph demands "faith and belief in eventual consistency." Success requires deliberate practice and operator expertise, not passive tool usage.

## Demonstrated Results

- Y Combinator hackathon: A coding agent shipped 6 repositories overnight
- Contract case study: A $50,000 USD project delivered for $297 USD using this method
- Language development: Ralph created a production-grade esoteric programming language entirely through iteration

## Key Applications

Ralph excels for greenfield projects and can replace substantial outsourcing work. It successfully generates code in languages not present in training data through continuous refinement cycles.
