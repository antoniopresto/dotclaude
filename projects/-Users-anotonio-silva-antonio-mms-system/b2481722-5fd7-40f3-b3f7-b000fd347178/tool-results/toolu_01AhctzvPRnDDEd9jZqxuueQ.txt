     1→# MMS Demo Application
     2→
     3→**Metro Fleet Mileage Management System** - Win-the-bid demonstration application.
     4→
     5→## Overview
     6→
     7→This is a demonstration application for Metro fleet preventive maintenance (PM) management. It showcases CSV-based PM projection over a 10-year horizon with an infinite canvas interface using semantic zoom levels (strategic → tactical → operational views).
     8→
     9→**Key Features:**
    10→- Drag-and-drop PM rescheduling with constraint validation
    11→- "Bow-wave" risk visualization (fleet maintenance clustering)
    12→- WebGL-powered infinite canvas rendering with deck.gl
    13→- State management with @belt/tools
    14→

    54→## @belt/tools Utilities Reference
    55→
    56→The `@belt/tools` package provides comprehensive utilities for state management, runtime validation, and common operations. All utilities are already available in this project.
    57→
    58→### State Machine
    59→
    60→Immer-based reactive state management with React hooks and automatic cache management.
    61→
    62→#### Creating Slices
    63→
    64→```typescript
    65→import { createSlice, createAsyncSlice, createMachine } from '@belt/tools';
    66→
    67→// Synchronous slice for local state
    68→const fleetSlice = createSlice({
    69→  id: 'fleet',
    70→  initial: {
    71→    trains: [],
    72→    pmHistory: [],
    73→    selectedTrain: null as string | null,
    74→  },
    75→  cache: { policy: 'session' }, // or 'public', 'private'
    76→  reducers: {
    77→    importFleet: (state, payload: Train[]) => {
    78→      state.trains = payload; // Immer allows direct mutations
    79→    },
    80→    selectTrain: (state, trainId: string) => {
    81→      state.selectedTrain = trainId;
    82→    },
    83→    clearSelection: (state) => {
    84→      state.selectedTrain = null;
    85→    },
    86→  },
    87→});
    88→
    89→// Async slice for data fetching
    90→const analyticsSlice = createAsyncSlice({
    91→  id: 'analytics',
    92→  cache: {
    93→    policy: 'public',
    94→    ttl: '5m', // Time-to-live using ms() format
    95→    swr: true, // Stale-while-revalidate
    96→  },
    97→  context: {},
    98→  load: async () => {
    99→    const response = await fetch('/api/analytics');
   100→    return response.json();
   101→  },
   102→});
   103→```
   104→
   105→#### Creating Machines
   106→
   107→```tsx
   108→import { createMachine } from '@belt/tools';
   109→
   110→const appMachine = createMachine({
   111→  id: 'mms',
   112→  slices: {
   113→    fleet: fleetSlice,
   114→    analytics: analyticsSlice,
   115→  },
   116→  context: { userId: null }, // Global context
   117→});
   118→
   119→// Use in React components
   120→function FleetView() {
   121→  const { state, actions } = appMachine.useMachine('fleet');
   122→
   123→  return (
   124→    <div>
   125→      <button onClick={() => actions.importFleet(trains)}>
   126→        Import Fleet
   127→      </button>
   128→      <div>{state.trains.length} trains loaded</div>
   129→    </div>
   130→  );
   131→}
   132→```
   133→
   134→**Cache Policies:**
   135→- `public` - Shared cache, TTL-based invalidation
   136→- `private` - User-specific cache, invalidated on user change
   137→- `session` - Session-specific cache, invalidated on app restart

   438→### Type Utilities
   439→
   440→Advanced TypeScript type helpers.
   441→
   442→```typescript
   443→import type {
   444→  Paths,          // All dot-notation paths of object
   445→  PathType,       // Type at specific path
   446→  Compute,        // Flatten intersection types
   447→  Writable,       // Remove readonly modifiers
   448→  Falsy,          // Union of falsy values
   449→  InferType,      // Infer type from schema
   450→  UnknownRecord,  // { [key: string]: unknown }
   451→  AnyRecord,      // { [key: string]: any }
   452→  MaybePromise,   // T | Promise<T>
   453→  MaybeArray,     // T | T[]
   454→  PromiseValue,   // Extract Promise<T> → T
   455→  Simplify,       // Flatten complex types
   456→  Merge,          // Deep merge two types
   457→  PartialFields,  // Make specific fields optional
   458→  NullableToPartial, // Convert nullable to optional
   459→  UnionToIntersection, // Union → Intersection
   460→  CastUnknown,    // Cast if unknown
   461→  SerializableOnly, // Only serializable fields
   462→} from '@belt/tools';
   463→
   464→// Example: Paths
   465→type User = {
   466→  id: string;
   467→  profile: {
   468→    address: {
   469→      street: string;
   470→    };
   471→  };
   472→};
   473→
   474→type UserPaths = Paths<User>;
   475→// 'id' | 'profile' | 'profile.address' | 'profile.address.street'
   476→
   477→type StreetType = PathType<User, 'profile.address.street'>;
   478→// string
   479→
   480→// Example: Compute (flatten intersections)
   481→type A = { a: number };
   482→type B = { b: string };
   483→type Flat = Compute<A & B>; // { a: number; b: string }
   484→
   485→// Example: Writable
   486→type ReadOnly = { readonly id: string; readonly name: string };
   487→type Mutable = Writable<ReadOnly>; // { id: string; name: string }
   488→
   489→// Example: PartialFields
   490→type Train = { id: string; name: string; metadata: object };
   491→type Optional = PartialFields<Train, ['metadata']>;
   492→// { id: string; name: string; metadata?: object }
   493→```
